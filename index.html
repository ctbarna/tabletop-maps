Tabletop.js Maps

This is a proof of concept using Tabletop.js as a data source for a map generated with the Google Maps API. The data on the map (when it loads) is of every crime that was reported in Chicago on September 11, 2011. The data was exported from Chicago's own data portal as CSV and then uploaded to Google Docs. All rows except those on 9/11/2011 were deleted as well as some columns. There have otherwise been no transformations of the data.

You can actually throw any published Google spreadsheet at this script and have it geocode the 'lat' and 'lng' or 'location' columns.
